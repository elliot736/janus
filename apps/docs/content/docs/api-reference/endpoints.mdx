## Public (used by SDK)

<Tabs items={["POST /challenge", "POST /verify"]}>

<Tab value="POST /challenge">

Request a proof-of-work challenge.

**Headers:** `X-Site-Key` (required)

**Response:**
```json
{
  "challengeId": "uuid",
  "challenge": "hmac-hash",
  "difficulty": 4,
  "mode": "invisible",
  "algorithm": "SHA-256",
  "expiresAt": "2026-03-22T10:05:00.000Z"
}
```

</Tab>

<Tab value="POST /verify">

Submit a solved challenge with collected signals.

**Headers:** `X-Site-Key` (required)

**Body:**
```json
{
  "challengeId": "uuid",
  "nonce": "48291",
  "solveTimeMs": 1847,
  "fingerprint": { "canvas": "...", "webgl": "..." },
  "behaviorData": { "mouseMovements": 23, "keystrokes": 5 }
}
```

**Response:**
```json
{
  "success": true,
  "token": "signed-token",
  "riskScore": 15,
  "action": "allow",
  "expiresAt": "2026-03-22T10:10:00.000Z"
}
```

</Tab>

</Tabs>

## Dashboard (authenticated)

<Accordions type="multiple">

<Accordion title="Sites">

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/v1/sites` | List all sites |
| POST | `/api/v1/sites` | Create a site |
| GET | `/api/v1/sites/:id` | Get site details |
| PUT | `/api/v1/sites/:id` | Update site |
| DELETE | `/api/v1/sites/:id` | Delete site |
| POST | `/api/v1/sites/:id/rotate-keys` | Rotate site keys |

</Accordion>

<Accordion title="API Keys">

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/v1/sites/:siteId/api-keys` | List API keys |
| POST | `/api/v1/sites/:siteId/api-keys` | Create API key |
| DELETE | `/api/v1/api-keys/:id` | Delete API key |

</Accordion>

<Accordion title="Analytics">

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/v1/analytics/:siteId/summary` | Summary stats |
| GET | `/api/v1/analytics/:siteId/requests-per-day` | Daily request counts |
| GET | `/api/v1/analytics/:siteId/risk-distribution` | Risk score buckets |
| GET | `/api/v1/analytics/:siteId/top-ips` | Top IPs by count |
| GET | `/api/v1/analytics/:siteId/stream` | SSE real-time feed |
| GET | `/api/v1/analytics/:siteId/adaptive-difficulty` | Current adaptive state |

</Accordion>

<Accordion title="Verification Logs & Data">

</Accordion>

</Accordions>
